=======================
Salt Telegraf formula
=======================

Power your metrics and alerting with a leading open-source monitoring
solution.

Sample pillars
==============

.. code-block:: yaml

  telegraf:
    agent:
      enabled: true
      interval: 15
      round_interval: false
      metric_batch_size: 1000
      metric_buffer_limit: 10000
      collection_jitter: 2
      output:
        prometheus_client:
	  prom1:
            bind:
              address: 0.0.0.0
              port: 9126
            engine: prometheus

Influx output
-------------

.. code-block:: yaml

  telegraf:
    agent:
      input:
        cpu:
          allcpus:
	    interval: 10
      output:
        influxdb:
	  influx1:
            urls:
              - http://127.0.0.1:8086
            database: test-telegraf
            write_consistency: any
            timeout: 10s
